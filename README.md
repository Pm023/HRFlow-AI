# HRflow AI - Professional HR Document Specialist

HRflow AI is a professional management platform designed to automate the generation of high-fidelity, legally appropriate corporate documents. 

## ✨ Key Features
- **Professional Persona**: Powered by logic derived from 15+ years of HR experience.
- **Glassmorphism UI**: A stunning, modern dark-mode interface.
- **Dynamic Preview**: Real-time rendering of documents.
- **Ready-to-Print**: Precision-formatted A4 output.

## 🚀 Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the server: `python main.py` or `python -m uvicorn app.main:app --port 8003`
